等差数列相关知识定义若数列({a_n})满足从第二项起，每一项与它的前一项之差为常数d，
递推公式：(a_{n + 1}-a_n = d(n\in N^))。当(d>0)时，({a_n})为递增数列；当(d = 0)时，({a_n})为常数数列；当(d<0)时，({a_n})为递减数列。通项公式① (a_n=a_1+(n - 1)d)，(n\in N^)，写作 (a_n=a_1+(n - 1)d)，(n\in N^)；
② (a_n=a_m+(n - m)d)，(m,n\in N^)，写作 (a_n=a_m+(n - m)d)，(m,n\in N^)；
③ (a_n=kn + b)（(k = d)，(b=a_1 - d)），此式是({a_n})为等差数列的充要条件，写作 (a_n=kn + b)（(k = d)，(b=a_1 - d)）。当(d = 0)时，(a_n)对应的函数(f(x))为常数函数；当(d\neq0)时，(a_n)对应的函数(f(x))为一次函数。性质① 若A，x，B成等差数列，则x为A与B的等差中项，且(x=\frac{A + B}{2})，写作 若(A)，(x)，(B)成等差数列，则(x)为(A)与(B)的等差中项，且(x = \frac{A + B}{2})；
② (a_n=\frac{a_{n - m}+a_{n + m}}{2})（(m,n\in N^)），写作 (a_n=\frac{a_{n - m}+a_{n + m}}{2})（(m,n\in N^)；
③ 若(m + n=p + q)（(m,n,p,q\in N^)），则(a_m+a_n=a_p+a_q) ，写作 若(m + n=p + q)（(m,n,p,q\in N^)），则(a_m + a_n=a_p + a_q)。前n项和(S_n=\frac{n(a_1 + a_n)}{2}=na_1+\frac{n(n - 1)}{2}d)，(n\in N^)，写作 (S_n=\frac{n(a_1 + a_n)}{2}=na_1+\frac{n(n - 1)}{2}d)，(n\in N*)。当(S_n=An2 + Bn)（(A=\frac{d}{2})，(B=a_1-\frac{d}{2})）时，首项(a_1=A + B)，公差(d = 2A)，写作 当(S_n=An^2 + Bn)（(A=\frac{d}{2})，(B=a_1-\frac{d}{2})）时，首项(a_1=A + B)，公差(d = 2A)。若数列({a_n})为等差数列，则数列({\frac{S_n}{n}})为等差数列，公差为(\frac{d}{2})，写作 若数列({a_n})为等差数列，则数列({\frac{S_n}{n}})为等差数列，公差为(\frac{d}{2})。设(S_n)为前n项和，则(S_n)，(S_{2n}-S_n)，(S_{3n}-S_{2n},\cdots)为等差数列，公差为(n^2d)，写作 设(S_n)为前(n)项和，则(S_n)，(S_{2n}-S_n)，(S_{3n}-S_{2n},\cdots)为等差数列，公差为(n^2d)。当项数为2n（(n\in N^*)）时：(S_{偶}=\sum_{i = 1}^{n}a_{2i}=n\cdot\frac{a_2 + a_{2n}}{2}=n\cdot a_{n + 1})，写作 (S_{偶}=\sum_{i = 1}^{n}a_{2i}=n\cdot\frac{a_2 + a_{2n}}{2}=n\cdot a_{n + 1})；(S_{奇}=\sum_{i = 1}^{n}a_{2i - 1}=n\cdot\frac{a_1 + a_{2n - 1}}{2}=n\cdot a_{n})，写作 (S_{奇}=\sum_{i = 1}^{n}a_{2i - 1}=n\cdot\frac{a_1 + a_{2n - 1}}{2}=n\cdot a_{n})；(S_{奇}-S_{偶}=-nd)，写作 (S_{奇}-S_{偶}=-nd)。当项数为(2n + 1)（(n\in N)）时：(S_{奇}-S_{偶}=a_{n + 1})，写作 (S_{奇}-S_{偶}=a_{n + 1})。
